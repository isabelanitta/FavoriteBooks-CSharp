Projeto pessoal dos meus estudos em Desenvolvimento de Sistemas!

Sistema interativo para organizar e exibir detalhes de quatro livros. Desenvolvido em C# (WinForms/WPF), com conceitos de Programação Orientada a Objetos (POO) e a criação de uma classe 'Livro' para a gestão de dados e também comunicação entre formulários via construtores, permitindo a passagem dinâmica de objetos.

OBS: As imagens das capas dos livros utilizadas neste projeto são de propriedade de suas respectivas editoras e foram utilizadas apenas para fins ilustrativos e educacionais.
